Manager Commands

Broadcaster, moderators, and trusted users can run these in Twitch chat.
#6help

Shows the quick manager command list in chat.

#6ping

Confirms the bot is connected and responding.

#6reload

Reloads command and role data from the server.

#6add !cmd response

Adds a new chat command with a default cooldown.

#6edit !cmd response

Updates the response for an existing command.

#6del !cmd

Deletes a command from chat and the dashboard.

#6enable !cmd

Turns a disabled command back on.

#6disable !cmd

Temporarily turns a command off.

#6cooldown !cmd 5

Changes command cooldown seconds.

#6level !cmd vip

Sets who can use a command.

#6alias !cmd !alias

Replaces aliases for a command.

#6info !cmd

Shows settings for a command.
